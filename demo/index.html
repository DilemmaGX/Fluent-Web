Buttons

Regular

Regular buttons acts just like any other buttons, but have a minimal width of 100px. It is also the default button type of this collection.


Compact

Compact buttons acts just like any other buttons, but with only necessary paddings, which means there are no minimal width. To activate compact mod, simply add "compact" into the class list.
